CT Page Editors allows you to add extra editable sections onto any custom page template.
== Description ==
Easiest and mostly configureless way to add additional WP wysiswg editors to any custom page template. 

== Installation ==
1. Upload "ct-page-editors" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Place "ctpe_content('Name of new editor');" anywhere in your page template that you wish to add a
